# Colocar o link do site no GitHub (seção "Sobre")

## Passo 1 — Ligar o site (obrigatório, 1 minuto)

1. Abra: https://github.com/victorhugoaurelianocoltro/ResolveAi/settings/pages
2. **Build and deployment → Source:** **Deploy from a branch**
3. **Branch:** `gh-pages` · pasta **`/ (root)`**
4. Clique **Save**
5. Aguarde 2–5 minutos e teste: https://victorhugoaurelianocoltro.github.io/ResolveAi/

## Passo 2 — Link visível na barra lateral (ícone da engrenagem)

1. Volte à página do repositório: https://github.com/victorhugoaurelianocoltro/ResolveAi
2. À direita, em **Sobre**, clique no **ícone de engrenagem** ⚙️
3. Preencha:
   - **Descrição:** `Emergência em casa em São Paulo — triagem grátis em 30s`
   - **Website:** `https://victorhugoaurelianocoltro.github.io/ResolveAi/`
   - (Opcional) marque **Releases** se quiser
4. **Save changes**

Qualquer pessoa que abrir o repositório verá o link do site em **Sobre**.
